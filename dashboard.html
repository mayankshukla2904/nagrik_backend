API Endpoints

Available Data Endpoints:

GET http://localhost:3000/api/complaints
Retrieve all complaints with pagination
GET http://localhost:3000/api/complaints/:id
Get specific complaint by ID
GET http://localhost:3000/api/dashboard/stats
Get dashboard statistics
GET http://localhost:3000/api/dashboard/analytics
Get detailed analytics data
GET http://localhost:3000/api/health
System health check
POST http://localhost:3000/api/complaints
Create new complaint

PowerShell Commands:

# Get all complaints
Invoke-RestMethod -Uri "http://localhost:3000/api/complaints" -Method GET
# Get dashboard stats
Invoke-RestMethod -Uri "http://localhost:3000/api/dashboard/stats" -Method GET
# Get system health
Invoke-RestMethod -Uri "http://localhost:3000/api/health" -Method GET
